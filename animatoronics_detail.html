Arduino/Processingで制御するアニマトロニクス

Animatronic figure controlled by Processing and Arduino

学部1年次 授業内自由課題にて発表

Presented as a project in a first-year undergraduate class



概要

Overview

所属大学のイメージキャラクターを模したロボットおよびその制御ソフトウェアを作成した。
ロボット内部は3Dプリンタで出力したフレームおよびサーボモータが設置されており、WindowsPC上で実行したソフトウェアからArduinoを介して動作する。

The system consists of a robot modeled after the University's image character and its control software.
The robot will have a 3D printed frame and servo motors, and will be controlled by software running on a Windows PC via Arduino.
