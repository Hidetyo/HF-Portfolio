Shadow Chamber: 3DCAD system that manipulates an object by inserting a hand between the object's image and its shadow

EC2023



Abstract

The system proposed in this paper consists of a transparent LCD display showing the image of an object and a touch panel display located behind it showing a virtual shadow of the object. The user inserts their hand between these displays and touches the virtual shadow with their fingertips. Touching the shadow deforms the object in conjunction with the shadow. This allows the system to be used as a 3DCAD system for designing the shape of a model.
